This project is a spring boot project.

Main Class: WordGenerationApplication.java


Steps to run project:

using cmd: /WordGeneration
build --> mvn clean install
run --> mvn spring-boot: run

This project has 1 custom exception. 
Exception handling is done using controller advice.

This project uses in-memory database.

While startup, all data from word_dictionary.txt file is inserted in in memory database.
